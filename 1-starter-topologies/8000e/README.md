# SRv6-Labs Starter Cisco 8000 Emulator Topologies

### Description: 
This directory contains instructions, configs, and topology files for launching a starter Cisco 8000 hardware-emulator topology with [Containerlab](https://containerlab.dev/). The sample topology is `clab-4-node-8000e.yml` (`name: 4-node-8000e`, `prefix: ""`). Deploy from this directory with `sudo containerlab deploy -t clab-4-node-8000e.yml`.
